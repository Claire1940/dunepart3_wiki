<Section icon={Target} title="『デューン 砂の救世主』の適応と主要な物語の構成">

『デューン 砂の救世主』を映画化するには、前2作の圧倒的な征服劇から、緊迫した心理的政治スリラーへとトーンを変化させる必要があります。**『デューン 砂の惑星PART3』のあらすじ**では、ポールのフレメンによる聖戦が宇宙全域に広がり、数十億人の犠牲者と、彼自身がもはや抑制できない宗教運動を引き起こしている様子が強調されます。

物語は主に3つの柱に焦点を当てています。

<CardGrid cols={3}>
  <Card title="帝国の陰謀">
    - **ベネ・ゲセリットの計画**: アトレイデスの血統を確保するための策動。
    - **ギルドの圧力**: メランジ（香料）生産を制御するための圧力。
    - **ベネ・トレイルァクスの計略**: 遺伝子操作を利用した陰謀。
  </Card>
  <Card title="家庭内の葛藤">
    - **ポールとチャニ**: 壊れてしまった愛の絆。
    - **皇女イルラン**: 政治的正当性を確保するための苦闘。
    - **アリア・アトレイデス**: 高まる権力と宗教的影響力。
  </Card>
  <Card title="過去の亡霊">
    - **ダンカン・アイダホの帰還**: トレイルァクスのゴーラ（複製人間）として。
    - **アトレイデス家の記憶**: 帝国の現実との衝突。
    - **伝説の剣士**: ポールの師の物理的な帰還。
  </Card>
</CardGrid>

前作から物語がどのように変化するかを理解するために、以下のテーマ別の比較を参考にしてください。

| 物語のテーマ | デューン 砂の惑星 PART2 | デューン 砂の惑星PART3（救世主） |
| :--- | :--- | :--- |

<Section icon={Swords} title="キャストとキャラクターガイド">

キャストリストには、受賞歴のある再登場俳優たちに加え、帝国の複雑な政治派閥を演じる注目度の高い新キャストが名を連ねています。ティモシー・シャラメとゼンデイヤがアンサンブルを牽引し、ロバート・パティンソンが主要な敵役としてシリーズに加わります。

```
帝国宮廷 (ポール、イルラン、アリア) <---> 陰謀者 (サイテイル、ベネ・ゲセリット)
       |                                           |
       +---------------> チャニ & フレメン <----------+
```

| 俳優 | キャラクター | 陣営 / あらすじにおける役割 |
| :--- | :--- | :--- |
| **ティモシー・シャラメ** | **ポール・アトレイデス** | 皇帝ムアディブ。予知能力に苦悩している。 |
| **ゼンデイヤ** | **チャニ** | フレメンの戦士。ポールの真実の愛であり、彼の統治の主な批判者。 |
| **ジェイソン・モモア** | **ダンカン・アイダホ** | ベネ・トレイルァクスによって送られたゴーラ「ハイト」として再生。 |
| **フローレンス・ピュー** | **皇女イルラン** | ポールの正妃。歴史に名を残すために画策する。 |
| **ロバート・パティンソン** | **サイテイル** | 謎めいたベネ・トレイルァクスを代表するフェイス・ダンサー。 |
| **アニャ・テイラー＝ジョイ** | **アリア・アトレイデス** | ポールの妹。強大なベネ・ゲセリットの能力を持つ。 |
| **レベッカ・ファーガソン** | **レディ・ジェシカ** | ポールの母。遠方からベネ・ゲセリットを導く。 |
| **ナコア＝ウルフ・モモア** | **レト2世** | ポールとチャニの幼い息子。ポールの父の名を冠している。 |
| **アイダ・ブルック** | **ガニマ** | レト2世の双子の妹。アラキスの未来において中心的な役割を担う。 |

<Callout type="info" title="キャラクター・スポットライト">
ジェイソン・モモアがトレイルァクスのゴーラ「ハイト」として帰還することは、ポールにとって極めて重要な感情的試練となります。ハイトは皇帝を暗殺するようにプログラムされていますが、ポールの愛した幼少期の師、ダンカン・アイダホの記憶を保持しています。
</Callout>

</Section>
